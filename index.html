Objectives

Our goal is to implement the following extensions to our current pathtracer implementation:

  • Realtime Rendering
  • Volumetric Lighting (implementation structure from insert link here)
  • Glass (refractive BRDFs)
  • a first-person demo
  • compatibility with .obj files

These can all help us with our final goal of rendering a realtime scene of a 1996 BMW M3

The volumetric lighting will help make the lights on the car more realistic, while glass will help make the car's windows see-through. Furthermore, we need to add .obj file compatibility in order to render the car model. Realtime rendering and the first person demo are extras feature that would be nice to have but are not necessary.
